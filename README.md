# EventosApp
Neste projeto feito durante o Entra21 para estudar Spring Boot, foi criada uma aplicação para que o usuário possa cadastrar seus eventos, com opção ainda de realizar login tanto como administrador quanto como usuário, cadastrar convidados, verificar os detalhes do evento desejado, editar qualquer evento e editar convidado.
